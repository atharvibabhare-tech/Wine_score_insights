# 🍷 Wine Rating Data Analysis Dashboard

## 📌 Project Overview
This project analyzes wine rating data to extract meaningful insights 
about rating distribution, premium segment dominance, and overall wine type trends.

## 📊 Features
- Total wine distribution by type
- Rating trend visualization
- Premium segment (90+ ratings) analysis
- Interactive file upload

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit

## 🚀 How to Run Locally

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## 📈 Key Insights
- Most wines fall in the 85–90 rating range.
- Premium wines (90+) show dominance of specific wine types.
- Distribution patterns vary significantly across wine categories.
